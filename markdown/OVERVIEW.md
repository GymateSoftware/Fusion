### Directories

```

├── docs/
│   ├── dist/
│   └── doxygen-awesome-css/
├── include/
├── markdown/
├── src/

```

- `docs`: Root directory of generated documentation
- `include`: C header files
- `markdown`: Markdown files used when generating docs
- `src`: C source files
- `vendor`: Third-party sources
